# BRIGHT-COFFEE-SHOP
# Project Overview

This project analyzes sales data from Bright Coffee Shop to uncover business insights that can help improve decision-making. The analysis covers sales performance, customer purchasing behavior, product performance, and revenue trends using SQL, Excel, and data visualization techniques.

The project demonstrates the complete data analytics workflow, from data cleaning and transformation to analysis and visualization.

# Project Objectives

* Clean and prepare raw sales data.
* Calculate key business metrics such as total revenue.
* Analyze monthly sales trends.
* Identify top-selling products.
* Evaluate sales by product category.
* Examine customer purchasing patterns.
* Create interactive charts and dashboards.
* Present actionable business recommendations.
  
# Tools & Technologies

* Microsoft Excel
* Databricks SQL
* GitHub
* Pivot Tables & Pivot Charts

# Dataset

The dataset contains transactional coffee shop sales data, including:

* Transaction Date
* Transaction Time
* Product Category
* Product Type
* Product Detail
* Quantity Sold
* Unit Price
* Total Revenue

# Analysis Performed

The project includes:

* Data cleaning
* Handling missing values
* Removing duplicates
* Data formatting
* Revenue calculations
* Monthly sales analysis
* Product performance analysis
* Category performance analysis
* Pivot tables
* Pivot charts
* Business insights and recommendations

# Key Business Questions

* What is the total revenue generated?
* Which month generated the highest sales?
* Which products contribute the most revenue?
* Which product categories perform best?
* How do sales change over time?
* What insights can help increase profitability?
  
# Key Insights

* Revenue varies across different months.
* A small number of products generate a significant share of total sales.
* Certain product categories consistently outperform others.
* Sales trends reveal opportunities for inventory planning and marketing campaigns.

 # Business Recommendations

* Increase inventory for top-selling products.
* Promote low-performing products through targeted marketing.
* Use seasonal trends to plan stock levels.
* Focus promotions during slower sales periods.
* Continue monitoring monthly performance using dashboards.
